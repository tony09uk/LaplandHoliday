Overview

Site introduction

Welcome to Lapland holiday. Your guide to everything you need to know to build your own trip to Lapland. This site is designed to answer all questions you have about building your dream trip to Lapland. If the answer you are looking for is not already here please get in touch with me. I started this site after organising a DIY lapland trip myself, for 9 people. I was so concerned I had missed something in the planning because I had to cobble bits of information from around the internet. So I have put all that information about travelling to lapland in one place for you. Please note that running this site costs me money, so I have added affiliate links to try and help cover those fees. Clicking these links will not cost you any money and will help the site to continue to run so if there is anything that you are interested in please use the links provided and checkout the adverts if they interest you. Any money made will go towards improving this site for future travellers.
